# Rododex Pátio

Sistema de medição de produtividade para conferentes de docas de carregamento e descarregamento.

## 📋 Funcionalidades

- Listagem de conferentes com cards individuais de produtividade
- Atualização automática dos dados a cada 6 segundos
- Integração com Google AppScript para persistência de dados
- Extração de dados do sistema SSW (via scraping/proxy)
- Interface responsiva com Ant Design
- Indicadores visuais de performance e metas

## 🚀 Tecnologias Utilizadas

- React 18
- Ant Design (ANTD)
- JavaScript (ES6+)
- Google AppScript (Backend improvisado)
- Vercel (Hospedagem)
